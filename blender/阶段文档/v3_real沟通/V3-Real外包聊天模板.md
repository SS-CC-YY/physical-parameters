# V3-Real 外包聊天模板

## 1. 第一条沟通消息

你好，我这里有一组 Blender 物理实验场景，需要把现有的纯净实验版本改造成更真实的室内/室外视觉场景。

我会提供每个实验的 `.blend` 文件和对应脚本。你需要做的是：在不改变物理运动轨迹的前提下，增加真实场景、材质、灯光、相机和道具，让画面更像真实拍摄视频。

重点要求：

1. 不允许修改实验物体的运动轨迹。
2. 不允许修改隐藏物理量定义。
3. 不允许因为场景建模导致物体穿模、遮挡关键事件或改变碰撞/落地/停止等关键时刻。
4. 你负责真实环境、真实材质、真实光照、真实镜头和道具包装。
5. 第一阶段先做 1 个验证样例，我确认风格和比例后，再继续做剩余场景。

## 2. 可以直接发给外包的任务说明

本项目要制作的是用于物理视频生成模型评测的 Blender 场景。底层物理轨迹已经由脚本严格控制，你不需要也不能重新设计物理运动。

请基于我提供的 `V3-Pure` 版本制作 `V3-Real` 版本。`V3-Real` 的目标是让场景更真实，但物理核保持不变。

你需要完成：

1. 根据纯净版实验添加真实场景背景。
2. 将理想几何场景包装成现实中合理存在的装置或环境。
3. 增加真实材质、贴图、灯光、阴影和相机景深。
4. 确保实验物体完整可见，关键物理事件清晰可见。
5. 保证所有物体之间不穿模。
6. 保证场景比例合理，不出现物体过大、家具过小、地面过厚等比例问题。

## 3. 不允许修改的部分

以下部分不能改：

1. 实验物体每一帧的位置和旋转关键帧。
2. 实验物体的运动阶段顺序。
3. 关键事件发生时刻，例如落地、碰撞、反弹、停止、通过阻尼区。
4. 隐藏物理量，例如 `g`、`mu`、`e`、`c`、`gamma`、`omega0` 等。
5. 实验主体的物理含义，例如球仍然必须是球，方块仍然必须是方块，弹簧仍然必须连接墙面和方块。

可以改：

1. 背景场景。
2. 材质和贴图。
3. 灯光。
4. 相机位置和焦距，但必须保证关键事件可见。
5. 非实验主体的装饰物和环境道具。

## 4. 验收标准

每个场景交付后，我会按以下标准验收：

1. `.blend` 文件可以正常打开。
2. 动画可以从第 1 帧播放到最后一帧。
3. 实验主体没有陷入地面、墙体、桌面或其他物体。
4. 关键事件不被遮挡。
5. 场景比例符合真实世界。
6. 材质和光照具有真实感。
7. 文件内对象、集合、相机和灯光命名清晰。
8. 所有外部贴图、HDR、模型资源都一并打包。

## 5. 推荐先做的验证样例

建议你先做一个验证样例，不要一次性做全部。

优先选择：

`L3-C`: 方块从斜面下滑，进入地面滑行，并与挡板碰撞反弹。

原因：

1. 这个实验同时包含斜面、地面和挡板，能快速检查真实场景包装能力。
2. 关键事件多，包括下滑、滑行、碰撞、反弹。
3. 如果这个样例能做好，其他实验大概率也能顺利推进。
