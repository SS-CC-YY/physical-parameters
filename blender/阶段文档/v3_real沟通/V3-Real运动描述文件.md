# V3-Real 运动描述文件

## 1. 核心原则

本文件用于向外包方说明：  
真实场景版本中，**只能替换视觉表现，不能修改底层运动定义**。

也就是说：

- 可替换：主体外观、场景、材质、背景、灯光、道具
- 不可替换：轨迹、接触顺序、隐藏物理量定义、关键事件时刻

---

## 2. 实验 A：Ball projectile + drag + bounce

### 运动阶段

1. 小球从初始位置以初速度抛出
2. 飞行过程中受重力和线性空气阻力
3. 与地面第一次接触后发生反弹
4. 后续继续飞行并逐渐停止

### 隐藏物理量

- `g_hidden`：重力加速度
- `c_hidden`：线性空气阻力系数
- `e_hidden`：地面恢复系数

### 外包实现要求

1. 不允许把球改成刚体模拟后重新求轨迹
2. 落地点、第一次反弹高度、后续衰减节律必须保持
3. 地面材质可变，但接触平面高度不可变

---

## 3. 实验 B：Block spring-damper oscillation

### 运动阶段

1. 方块从偏离平衡位置的状态开始
2. 在水平导轨上来回振动
3. 振幅随时间衰减
4. 弹簧应当随方块整体伸缩

### 隐藏物理量

- `omega0_hidden`：无阻尼固有频率
- `gamma_hidden`：阻尼系数

### 外包实现要求

1. 弹簧必须是连续整体，不是离散不动圆环
2. 方块始终与导轨接触，不允许悬空或嵌入
3. 方块的平衡点、振幅、周期与阻尼包络必须与 Pure 版一致
4. 弹簧两端要有明确锚定点：一端连固定墙面或支架，另一端连方块侧面

---

## 4. 实验 C：Block slide + wall collision

### 运动阶段

1. 方块从斜面顶部开始下滑
2. 在斜面上受重力和摩擦共同作用加速
3. 离开斜面后进入水平地面继续减速滑行
4. 与挡板发生碰撞
5. 以较低速度反向运动并停止

### 隐藏物理量

- `g_hidden`：重力加速度
- `mu_hidden`：摩擦系数
- `e_hidden`：挡板恢复系数

### 外包实现要求

1. 挡板位置不可任意改动
2. 挡板与方块碰撞时不能穿模
3. 方块中心高度在斜面和地面阶段必须严格贴合接触边界
4. 不允许删掉碰撞段或把碰撞改成停住不反弹

---

## 5. 实验 D：Ball pendulum + localized damping zone

### 运动阶段

1. 小球通过细杆连接到固定支点
2. 小球做摆动
3. 当摆球经过底部局部阻尼区时，能量显著损失
4. 阻尼不是全程均匀的，而是“过区时增强”

### 隐藏物理量

- `g_hidden`：重力加速度
- `gamma_zone_hidden`：局部阻尼区阻尼强度

### 外包实现要求

1. 支点位置固定
2. 细杆上端固定在支点，下端固定在球体上
3. 局部阻尼区应当被可视化，例如磁区、阻尼帘区、局部电磁区
4. 不允许把整个摆替换成全程均匀阻尼模型

---

## 6. 统一的关键事件检查点

外包版本必须能清楚看到以下关键事件：

### A

- 起抛
- 第一次落地
- 第一次反弹

### B

- 初始最大偏移
- 若干个往返峰值
- 衰减后接近平衡

### C

- 从斜面开始下滑
- 离开斜面进入地面
- 与挡板接触
- 反向回弹

### D

- 初始摆动峰值
- 多次穿过底部阻尼区
- 后期显著衰减

---

## 7. 推荐给外包方的工作流程

1. 先导入或对齐甲方给定 Pure 版轨迹
2. 再搭真实场景
3. 再替换主体外观
4. 再微调相机和灯光
5. 最后逐帧检查：
   - 是否穿模
   - 是否漏拍关键事件
   - 是否改变轨迹

---

## 8. 最小验收标准

一个外包版本至少要通过以下检查：

1. 主体轨迹与 Pure 版一致
2. 关键物理事件全部在画面中
3. 无穿模
4. 无错误比例
5. 弹簧、摆杆、斜面、挡板等关键结构具有真实连接关系
