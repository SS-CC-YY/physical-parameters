# V3 实验对照表

## 版本定位

V3 对应“纯净多参数耦合”层级。与 V2 不同，V3 中每个实验都至少包含 2 个可从轨迹反推的隐式物理量，并且强调多物理规律共同作用下的参数分解能力。

## 与 V2 的区别

- V2 是“单参数 + 长时序”。
- V3 是“多参数 + 多规律耦合”，并进一步覆盖滚动、双体碰撞和空间非均匀物理场。

因此，V3 的重点不再是更长的轨迹，而是模型能否把多个物理因素从同一条轨迹中分离出来。
